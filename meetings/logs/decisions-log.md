# Decision Log - Maroa Pilot

> Registro historico de todas las decisiones tomadas en reuniones.
> Cada decision tiene un ID unico para referencia.

---

## Como Usar

1. Despues de cada reunion, agregar decisiones aqui
2. Usar formato de ID: `DEC-YYYYMMDD-##`
3. Vincular a la reunion donde se tomo
4. Marcar estado: **Activa** / **Superseded** / **Reverted**

## Template para Nueva Decision

```markdown

**Fecha:** YYYY-MM-DD
**Reunion:** [Link a notas]
**Participantes:** [Quienes decidieron]

**Contexto:**
[Por que se necesitaba esta decision]

**Opciones Consideradas:**
1. Opcion A - [pros/cons]
2. Opcion B - [pros/cons]

**Decision:**
[Que se decidio]

**Razon:**
[Por que se eligio esta opcion]

**Impacto:**
- Documentos afectados: [lista]
- Tareas derivadas: [lista]

**Estado:** Activa / Superseded by DEC-XXX / Reverted
```
